🎬 Projeto IMDB - Análise de Filmes

📌 Descrição

Este projeto tem como finalidade analisar dados de filmes utilizando o Power BI. Ao longo do desenvolvimento, foram aplicados processos de ETL (Extração, Transformação e Carga), modelagem de dados e criação de dashboards interativos, com o objetivo de gerar insights relevantes a partir das informações.

❓ Perguntas de Negócio

- Qual é a quantidade total de filmes no dataset? → 31,8 mil  
- Qual a média das avaliações do IMDB para filmes lançados após 2000? → 58,43  
- O resultado financeiro da indústria foi positivo ou negativo? → Prejuízo de -316 bilhões  
- Quais gêneros apresentam maior geração de receita? → Action Epic, Action e Animal Adventure
